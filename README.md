# PowerBi-Global-Airbnb-Performance-Dashboard
Developed an interactive Airbnb analytics dashboard using historical data to uncover trends, monitor KPIs, and support data-driven decision-making.
Implemented advanced DAX measures to analyze Superhost listings, calculate cumulative percentages by city, and evaluate average price by room type through dynamic visualizations and insights.
Utilized Power BI bookmarks to optimize dashboard space, enabling multiple visualizations and improving user navigation and report interactivit.
